# landing-page_PT_MSN

## Deskripsi
Project ini merupakan website landing page sederhana yang dibuat menggunakan HTML dan CSS. Project ini bertujuan untuk menampilkan tampilan awal website perusahaan PT Media Solusi Network.

## Anggota Kelompok
- Alvin
- Azzam
- Thomi 

## Teknologi yang Digunakan
- HTML
- CSS

## Cara Menjalankan Project
1. Download atau clone project ini.
2. Buka folder project.
3. Jalankan file `index.html` menggunakan browser.

## Struktur Folder
```
landing-page_PT_MSN/
│
├── assets/
│   └── (gambar, icon, dll)
│
├── css/
│   └── style.css
│
├── index.html
├── .gitignore
└── README.md
```

## Fitur
- Tampilan Landing Page
- Responsive Layout
- Menggunakan HTML dan CSS

## Tujuan Project
Project ini dibuat sebagai latihan pembuatan website sederhana sekaligus untuk mempelajari struktur project HTML dan CSS serta penggunaan Git dan GitHub.
